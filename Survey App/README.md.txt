# 👧🧒 Children's Survey Form

A fun and friendly HTML form designed to collect basic personal and preference information from children. This can be used in schools, kids' programs, or web projects that involve early learners.

## 🌟 Features

- Simple and colorful UI
- Inputs for name, age, gender, hobbies, favorite color, and more
- Child-friendly layout
- Easy to run locally in a browser

## 🖼 Screenshot / Banner

![Survey Banner](child.jpg)

## 🧪 Technologies Used

- HTML5
- Basic inline CSS (optional for styling)

## 🚀 How to Use

1. Clone or download this repository
2. Open the `survey.html` file in any web browser (Chrome, Edge, Firefox, etc.)
3. Fill out the form and view the structured layout
